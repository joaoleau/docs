
## S3 Analytics

O Amazon S3 Analytics Storage Class Analysis ajuda a analisar padrões de acesso para otimizar a transição de dados entre classes de armazenamento.

~={red}Não é capaz de identificar acessos não intencionais ou problemas de segurança em buckets S3.=~
## S3 Object Lock

O S3 Object Lock permite armazenar objetos usando o modelo ~={green}"Write Once Read Many" (WORM)=~, prevenindo exclusões acidentais ou inadequadas de dados.
